Estrutura básica do HTML

HTML (HyperText Markup Language) é a linguagem padrão utilizada para criar a estrutura de páginas da web. Imagine-a como os ossos de uma casa: ela define a base, a organização e a hierarquia do conteúdo.

Componentes essenciais:

  • DocType: A primeira linha de um documento HTML declara o tipo de documento. Indica ao navegador qual versão do HTML está sendo utilizada.
  • <html></html>: É a tag raiz, englobando todo o conteúdo da página.
  • <head></head>: Contém informações sobre a página que não são exibidas diretamente no navegador. Aqui você encontra o título da página, metadados, links para folhas de estilo (CSS) e scripts (JavaScript).
  • <body></body>: É onde reside o conteúdo visível da página, como texto, imagens, links, tabelas, etc.

Exemplo simples:

                    <!DOCTYPE html>
                    <html>
                    <head>
                        <title>Minha Primeira Página</title>
                    </head>
                    <body>
                        <h1>Bem-vindo ao meu site!</h1>
                        <p>Este é um parágrafo de exemplo.</p>
                    </body>
                    </html>

Explicando cada parte:

  • <!DOCTYPE html>: Declara que este é um documento HTML5.
  • <title></title>: Define o título da página, que aparece na aba do navegador.
  • <h1></h1>: Cria um título de nível 1. Existem outros níveis (<h2>, <h3>, etc.) para títulos menos importantes.
  • <p></p>: Define um parágrafo.

Em resumo: O HTML é a base de toda página web, fornecendo a estrutura e organização necessárias para a criação de conteúdo online. Ao entender os conceitos básicos, você estará pronto para dar seus primeiros passos no desenvolvimento web.

O que são Tags

As tags HTML são como palavras-chave que informam ao navegador como interpretar e exibir o conteúdo de uma página. Elas delimitam elementos, como títulos, parágrafos, imagens, links, etc., e definem suas características.

Estrutura básica de uma tag:

                    <tag_name>Conteúdo da tag</tag_name>

Tipos de Tags e seus usos:

  • Tags Semânticas: Definem o significado do conteúdo. Exemplos:

    • <h1> a <h6>: Títulos de diferentes níveis de importância.
    • <p>: Parágrafo.
    • <nav>: Seção de navegação
    • <article>: Conteúdo independente (artigo, post).
    • <section>: Seção de conteúdo.

  • Tags Estruturais: Definem a estrutura da página. Exemplos:

    • <div>: Divisão genérica para agrupar outros elementos.
    • <span>: Contêiner inline para agrupar elementos em linha.
    • <header>: Cabeçalho da página.
    • <footer>: Rodapé da página.

  • Tags para Formatação: Aplicam estilos básicos ao texto. Exemplos:

    • <b> e <strong>: Texto em negrito (sem semântica e com semântica).
    • <i> e <em>: Texto em itálico (sem semântica e com semântica).
    • <ul> e <ol>: Listas não ordenadas e ordenadas.
    • <a>: Link para outra página ou ancoragem na mesma página.

  • Tags para Mídia: Inserem imagens, vídeos e outros tipos de mídia. Exemplos:

    • <img>: Imagem.
    • <video>: Vídeo.
    • <audio>: Áudio.

Exemplos básicos:

                    <h1>Título Principal</h1>
                    <p>Este é um parágrafo.</p>
                    <a href="https://www.exemplo.com">Link para outro site</a>
                    <ul>
                        <li>Item 1</li>
                        <li>Item 2</li>
                    </ul>
                    <img src="minha_imagem.jpg" alt="Descrição da imagem">

O que são Atributos

Atributos são palavras-chave que você adiciona dentro de uma tag de abertura para fornecer informações adicionais sobre aquele elemento. Eles permitem que você customize a aparência, o comportamento e outras características do elemento.

Sintaxe:

                    <tag_name atributo1="valor1" atributo2="valor2">Conteúdo</tag_name>
  • atributo: Nome do atributo.
  • valor: Valor atribuído ao atributo, sempre entre aspas.

Funções dos Atributos:

  • Especificar propriedades: Definir o tamanho, a cor, a posição, etc., de um elemento.
  • Fornecer informações adicionais: Dar mais contexto sobre o elemento, como o endereço de um link ou o tipo de um input.
  • Controlar o comportamento: Alterar a forma como um elemento interage com o usuário ou com outras partes da página.

Exemplos Comuns de Atributos:

  • id: Atribui um identificador único a um elemento.
  • class: Atribui uma ou mais classes a um elemento, permitindo aplicar estilos CSS.
  • src: Especifica o endereço de um recurso externo, como uma imagem ou um arquivo de áudio.
  • href: Define o destino de um link.
  • alt: Fornece um texto alternativo para uma imagem, utilizado por leitores de tela.

Exemplo no HTML:

                    <img src="minha_imagem.jpg" alt="Uma bela paisagem">
                    <a href="https://www.exemplo.com" target="_blank">Visite nosso site</a>
                    <input type="text" placeholder="Digite seu nome">

Atributos Globais e Específicos:

  • Atributos globais: Podem ser usados em quase todos os elementos, como id, class, title, style, etc.
  • Atributos específicos: São exclusivos de determinados elementos, como src para a tag <img> ou type para a tag <input>.

Por que usar atributos?

  • Personalização: Permite criar elementos únicos e personalizados.
  • Interatividade: Cria elementos que respondem a interações do usuário.
  • Acessibilidade: Melhora a acessibilidade da página para pessoas com deficiência.

Atributos e CSS

Os atributos id e class são especialmente importantes para aplicar estilos CSS aos elementos. Ao atribuir uma classe a um elemento, você pode definir um conjunto de regras CSS que se aplicam a todos os elementos com aquela classe.

Exemplos no HTML e CSS:

                    HTML
                    <h1 class="titulo">Título da Página</h1>
                    Use code with caution.
                    
                    CSS
                    .titulo {
                      color: blue;
                      font-size: 36px;
                    }

O que são textos no HTML

O texto é um dos elementos mais básicos e importantes de qualquer página web, e o HTML oferece diversas tags e atributos para formatá-lo e organizá-lo de forma eficiente.

Estrutura Básica de Texto em HTML:

  • Tags de parágrafo:

    • <p>: Define um parágrafo. É a tag mais comum para organizar o texto em blocos.

  • Quebras de linha:

    • <br>: Insere uma quebra de linha dentro de um parágrafo.

  • Formatação básica:

    • <b>: Texto em negrito.
    • <i>: Texto em itálico
    • <u>: Texto sublinhado.
    • <mark>: Texto marcado (geralmente com fundo amarelo).
    • <del>: Texto deletado (geralmente riscado).
    • <ins>: Texto inserido (geralmente sublinhado).

Organizando o Texto:

  • Cabeçalhos:

    • <h1> a <h6>: Definem títulos de diferentes níveis de importância.

  • Listas:

    • <ul>: Lista não ordenada (com marcadores).
    • <ol>: Lista ordenada (numerada).
    • <li>: Item de uma lista.

  • Citações:

    • <q>: Citações curtas.
    • <blockquote></blockquote>: Citações longas, geralmente com recuo.

Atributos Úteis para Texto:

  • style: Aplica estilos CSS diretamente ao elemento.
  • class: Atribui uma classe ao elemento para aplicar estilos através de CSS.
  • title: Adiciona um texto de dica que aparece quando o mouse passa sobre o elemento.

Exemplo prático:

                    <h1>Título Principal</h1>
                    <p>Este é um parágrafo de exemplo. <b>Este texto está em negrito.</b></p>
                    <ul>
                        <li>Item 1</li>
                        <li>Item 2</li>
                    </ul>
                    <p>Aqui temos uma citação: <q>A prática leva à perfeição.</q></p>

Por que a Formatação de Texto é Importante?

  • Legibilidade: Torna o conteúdo mais fácil de ler e entender.
  • Hierarquia: Cria uma hierarquia visual para o conteúdo, destacando os pontos mais importantes.
  • Acessibilidade: Facilita a leitura por pessoas com deficiência visual, utilizando leitores de tela.

Formatação Avançada com CSS

Embora o HTML ofereça algumas opções básicas de formatação, o CSS (Cascading Style Sheets) é a linguagem ideal para criar estilos mais complexos e personalizados para seus textos. Com CSS, você pode controlar a cor, o tamanho da fonte, o espaçamento entre linhas, a alinhamento, e muito mais.

Listas Ordenadas e Não Ordenadas

Listas são elementos essenciais em HTML para apresentar informações de forma organizada e hierárquica. Elas são especialmente úteis para exibir conjuntos de itens relacionados, como instruções, listas de produtos, etc.

Existem dois tipos principais de listas em HTML:

  • Listas não ordenadas <ul>: Utilizadas quando a ordem dos itens não é relevante. Os itens são marcados com marcadores (bullets) por padrão.
  • Listas ordenadas <ol>: Utilizadas quando a ordem dos itens é importante. Os itens são numerados.

Estrutura Básica:

                    <ul>
                        <li>Item 1</li>
                        <li>Item 2</li>
                        <li>Item 3</li>
                    </ul>
                      
                    <ol>
                        <li>Primeiro item</li>
                        <li>Segundo item</li>
                        <li>Terceiro item</li>
                    </ol>
  • <ul>: Define uma lista não ordenada.
  • <ol>: Define uma lista ordenada.
  • <li>: Define um item dentro da lista.

Personalizando Listas:

  • Tipos de marcadores: Você pode personalizar o tipo de marcador em listas não ordenadas usando CSS.
  • Tipos de numeração: Em listas ordenadas, você pode alterar o tipo de numeração (arábicos, romanos, letras, etc.) usando CSS.
  • Níveis de aninhamento: Crie listas dentro de listas para organizar informações mais complexas.

Exemplo com Níveis de Aninhamento:

                    <ul>
                        <li>Frutas:
                            <ul>
                                <li>Abacate</li>
                                <li>Abacaxi</li>
                            </ul>
                        </li>
                        <li>Legumes:
                            <ul>
                                <li>Abóbora</li>
                                <li>Abobrinha</li>
                            </ul>                        
                        </li>
                        <li>Grãos:
                            <ul>
                                <li>Arroz</li>
                                <li>Feijão</li>
                                <li>Milho</li>
                            </ul>
                        </li>
                    </ul>

Por que Usar Listas?

  • Organização: : Melhora a legibilidade e a compreensão do conteúdo.
  • Hierarquia: Permite criar estruturas hierárquicas para informações relacionadas.
  • Acessibilidade: Facilita a navegação por leitores de tela.

Atributos Úteis:

  • type: (Apenas para <ol>) Define o tipo de numeração (A, a, I, i, 1).
  • start: (Apenas para <ol>) Define o número inicial da contagem.

Entendendo links

Os links permitem que os usuários naveguem de uma página para outra, tornando a internet uma rede interconectada de informações.

A Tag <a>: A Base dos Links

A tag <a> (anchor) é a responsável por criar links em HTML. O atributo mais importante dessa tag é o href, que especifica o endereço (URL) para o qual o link irá direcionar.

Sintaxe básica:

                    <a href="url">Texto do link<;a>
  • href: O endereço do destino.
  • Texto do link: O texto visível que o usuário clica.

Exemplo:

                    <a href="https://www.google.com">Visite o Google<;a>

Tipos de Links:

  • Links internos: Conectam a página atual a outras páginas dentro do mesmo site.
  • Links externos: Conectam a página atual a páginas em outros sites.
  • Links para âncoras: Conectam a página atual a uma seção específica da mesma página.
  • Links para arquivos: Conectam a página atual a arquivos para download (PDF, documentos, etc.).
  • Links para e-mail: Criam um link que, quando clicado, abre um programa de e-mail para enviar uma mensagem.

Atributos Úteis para Links:

  • target: Define onde o link será aberto.
  • _blank: Abre em uma nova aba.
  • _self: Abre na mesma aba (padrão).
  • _parent: Abre no frame pai.
  • _top: Abre na janela principal.
  • title: Adiciona um texto de dica que aparece quando o mouse passa sobre o link.
  • rel: Especifica a relação entre a página atual e o recurso vinculado (usado para SEO e meta tags).

Exemplo com atributos:

                    <a href="contato.html" target="_blank" title="Entre em contato conosco">Contato</a>

Por que os Links são Importantes?

  • Navegação: Permitem que os usuários naveguem entre as páginas de um site.
  • Interconexão: Conectam sites diferentes, formando a World Wide Web.
  • SEO: Ajudam os motores de busca a encontrar e indexar seu conteúdo.
  • Experiência do usuário: Facilitam a navegação e a descoberta de informações.

Links e Acessibilidade

  • Texto alternativo: Utilize o atributo alt para fornecer um texto alternativo para imagens que funcionam como links, tornando o site mais acessível para pessoas com deficiência visual.
  • Foco do teclado: Certifique-se que os links sejam navegáveis pelo teclado, para usuários que não utilizam o mouse.
